# SCORM Review App (Next.js Application)

This is the front-end application for the SCORM Review App. It is built with [Next.js](https://nextjs.org/), React, and TypeScript. It utilizes libraries like `formidable` and `jszip` for handling SCORM package processing.

## Setup and Installation

1. Ensure you have Node.js installed.
2. Open a terminal and navigate to this specific application directory:
   ```bash
   cd /Users/oniesenpai/desktop/scorm-review-app/scorm-review-app
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## How to Run the Application

This project uses standard Next.js scripts configured in `package.json`.

### Development Server

To run the application locally for development (with hot-reloading):

```bash
npm run dev
```

After starting the server, open your browser and navigate to [http://localhost:3000](http://localhost:3000) (or whichever port Next.js assigns).

### Production Build

To build the application for production:

```bash
npm run build
```

To start the production server after building:

```bash
npm run start
```

### Linting

To run the linter and check for any code issues:

```bash
npm run lint
```
